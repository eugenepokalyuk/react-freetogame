## About
Developed the interface for the Free-To-Play Games website complete with a full layout of the Main Page of the website.

## Stack is
### Frontend
<ul>
  <li>React</li>
  <li>TypeScript</li>
  <li>Redux</li>
</ul>

### Backend
<ul>
  <li>NodeJS</li>
  <li>Express</li>
</ul>

## Scripts
Frontend:

- `build` - build for production

Frontend:
